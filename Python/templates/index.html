Depression Calculator

Depression Calculator (depr-calc) is a platform that detects the level of potential depression. Here, you can diagnose how likely you are currently experiencing depression. The application works using the Learning Vector Quantization method, which predicts based on data such as age, gender, and the characteristics or symptoms that you can input in the form below.
